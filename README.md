# Promeraco — Brand Boost

Marketing site for Promeraco (ad marketing, brand scaling, web design).

## Development

```bash
npm install
npm run dev
```

Dev server: `http://localhost:8080` (see `vite.config.ts`).

## Build

```bash
npm run build
```

Output in `dist/`. Deploy the `dist` folder (e.g. Vercel with framework preset Vite).
